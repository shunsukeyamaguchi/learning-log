### 概要
JavaScriptの基礎学習として、学習内容を管理できるログアプリを作成しました。
ログの追加・編集・削除（CRUD）に対応し、localStorageを利用してデータを永続化しています。

### 実装した機能
- 学習ログの追加
- 学習ログの編集（入力フォームを使った更新）
- 学習ログの削除
- localStorageによるデータ保存（リロード後も保持）
- 配列（状態）を元にDOMを再描画する設計

### 学んだこと
- 配列をアプリの状態として管理する考え方
- DOMは状態の結果として描画するという設計思想
- indexを使った編集・削除処理
- JSON.stringify / JSON.parse を使った永続化
- 「表示」と「データ操作」を分離する重要性
